- Você usa um agente de código e ele inventa especificação
- Agente fala 'corrigido' sem validar
- Agente expande escopo sem pedir
- Quer disciplina pra delivery sem retrabalho

## Entregáveis
- System prompt pronto pra colar
- Checklist mínimo antes de toda entrega
- Anti-patterns explícitos
- Casos exemplo end-to-end
- Estrutura de mensagem padronizada
